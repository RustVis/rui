- [Documentation](https://docs.rs/zu-util)
- [Release notes](https://github.com/RustVis/zu-util/releases)

## Usage
Add this to `Cargo.toml`:
```toml
[dependencies]
zu-util = "0.3"
```

Or using `cargo add zu-util` command.
