
# About
Wrapper of bootstrap icons for yew framework.

## How to use

First add this to Cargo.toml:

```toml
[dependencies]
zuicon-bs = "0.3"
```

Then import specific icons in yew components, like:
```rust
use zuicon_bs::{Mailbox, Map};
```
