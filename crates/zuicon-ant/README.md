
# About

zuicon-ant is based on [Ant Design SVG Icons](https://github.com/ant-design/ant-design-icons).

## How to use

First add this to `Cargo.toml`:
```toml
[dependencies]
zuicon-ant = "0.3"
```

Then import specific icons in yew components:
```rust
use zuicon_ant::filled::{Home, Mail};
```
