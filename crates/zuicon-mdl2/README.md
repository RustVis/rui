
# About

Migrate of Fluentui Icons (MDL2) to yew framework.

## How to use

First add this to `Cargo.toml`:

```toml
[dependencies]
zuicon-mdl2 = "0.3"
```

Then import specific icons in yew components:
```rust
use zuicon_mdl2::{Home, PublicEmail};
```
